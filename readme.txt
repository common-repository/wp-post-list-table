WordPress post or custom post type listing table.

== Description ==


WordPress Post List Table plugin will allow to make a listing of post inside table. You can show custom post type like WooCommerce products or any kind of post type. Post title, category, tag, date, author, comment, WooCommerce price, SKU, stock, review, add to cart etc can be shown in the table.


>[DEMO](http://demo1.wpbean.com/post-list-table/)  |  [Support](https://wpbean.com/support/) |  [Details Documentation](https://wpbean.com/wp-post-list-table-documentation/)


### Plugin Features:

* Post list table.
* Custom post type can be shown.
* WooCommerce support.
* Three different style for table.
* Light weight & customizable.

== Installation ==

* Install it as a regular WordPress plugin

* Create a page using this plugin's shortcode.

* Go to this plugin setting for customize it. 

### ShortCode 

[wpb_post_list_table]
